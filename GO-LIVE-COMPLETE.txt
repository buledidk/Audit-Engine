🔧 TROUBLESHOOTING & SUPPORT
═══════════════════════════════════════════════════════════════════════════════════

IF MODELS DON'T CONNECT:
  1. Check ANTHROPIC_API_KEY is valid
  2. Check OPENAI_API_KEY is valid
  3. System will fallback to Ollama (always available)
  4. Run: npm run verify:prod

IF CONNECTORS DON'T CONNECT:
  1. Check env vars are set
  2. Run: npm run verify:prod
  3. View logs: docker-compose logs audit-engine
  4. System continues without failed connectors

IF DATABASE CONNECTION FAILS:
  1. Verify DATABASE_URL format
  2. Check PostgreSQL is running
  3. Test: psql $DATABASE_URL -c "SELECT 1"
  4. Run migrations if needed

IF HEALTH CHECKS FAIL:
  1. Run full verification: npm run verify:prod
  2. Check individual component logs
  3. Ensure all services are started
  4. Wait 10-30 seconds for services to initialize
