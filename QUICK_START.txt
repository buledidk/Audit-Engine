================================================================================
  TROUBLESHOOTING
================================================================================

Port 3000 in use?
  kill -9 $(lsof -t -i :3000)
  npm run dev

Tests failing?
  npm run lint          # Check syntax
  npm run type-check    # Check types
  npm run test          # Run tests
  Fix errors and save → tests auto-run

Build failing?
  npm run lint -- --fix  # Auto-fix issues
  npm run build          # Try again
