```javascript

Professional Interpretation:
```

#### C. Control Risk Assessment (ISA 330)
```javascript
Control Risk (CR) =
  (DesignEffectiveness × 0.5 +
   OperatingEffectiveness × 0.5)

Design Testing:
  ✓ Control exists and is properly designed
  ✓ Control can operate as designed
  ✓ All assertions supported

Operating Testing:
  ✓ Control operated throughout period
  ✓ No significant deviations
  ✓ Deviation rate < threshold

Deviation Analysis:
  Deviation Rate = Exceptions / Sample Size
  Projected = (Exceptions / Sample) × Population
```

#### D. Detection Risk Determination (ISA 330)
```javascript
Detection Risk (DR) =
  Acceptable DR Target / (IR × CR)

Where:
  Acceptable DR Target = 5% (low) or 15% (medium)
  IR = Inherent Risk (0-1)
  CR = Control Risk (0-1)

Example:
  IR = 0.8 (HIGH), CR = 0.4 (MEDIUM)
  DR = 0.05 / (0.8 × 0.4) = 0.156 (15.6% Required)

  Interpretation:
  - High IR & medium CR = LOW acceptable DR
  - Requires EXTENSIVE substantive testing
  - Sample size increases 40-60%

This is EMBEDDED in: AUDIT_PROC_AGENT
```
