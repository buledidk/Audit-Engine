# Audit Quality Reference Reports

for the AuditEngine regulatory framework. PDFs are gitignored due to size (~18MB).

## Usage

Place PDF files in this directory. They are referenced by the `RegulatoryUpdateEngine`
and `AuditAccuracyEnhancementEngine` for audit quality benchmarking.
